## 動画スクリプト生成

ショート動画用のテロップとナレーション原稿を作成する。

### Step 1: 動画ファイルを特定する

!`ls "/Users/keeee/Desktop/Dev/Paper/作業中動画/" 2>/dev/null`

上記の結果を確認する:
- 動画ファイルが **1本** → そのファイルを自動選択して処理を続行する
- 動画ファイルが **複数本** → ファイル一覧をユーザーに提示し「どの動画を処理しますか？」と確認する
- ファイルが **0本** → 「作業中動画フォルダに動画ファイルが見つかりませんでした」と伝えて終了する

以降の `VIDEO_PATH` は選択されたファイルのフルパス（例: `/Users/keeee/Desktop/Dev/Paper/作業中動画/xxx.mp4`）とする。
また、`VIDEO_NAME` は拡張子を除いたファイル名（例: `scene1_商品フォーカス`）とする。

### Step 2: フレーム抽出

選択した動画からフレームを抽出する（3秒に1枚）。`VIDEO_PATH` を実際のパスに置き換えてBashで実行:

```
mkdir -p /tmp/video-script-frames && rm -f /tmp/video-script-frames/*.jpg && ffmpeg -i "VIDEO_PATH" -vf "fps=1/3,scale=640:-1" /tmp/video-script-frames/frame-%03d.jpg -y 2>/dev/null && ls /tmp/video-script-frames/
```

### Step 3: フレーム分析

`/tmp/video-script-frames/` 内の全 .jpg ファイルを Read ツールで読み込み、以下を分析する:
- 動画のジャンル（美容、健康、ファッション、食品、ライフスタイル等）
- 映っている商品・人物・シーン・雰囲気

### Step 4: スクリプト生成・保存

分析結果をもとにスクリプトを生成し、Write ツールで以下のパスに保存する:

保存先: `/Users/keeee/Desktop/Dev/Paper/作業中動画/テロップとナレーション.md`

保存する内容のフォーマット:

```
# [VIDEO_NAME] テロップ・ナレーション原稿

## 【フック】
**テロップ:** （[ジャンル]に関する驚きの事実。15文字以内）
**ナレーション:** （フックを補足する1〜2文。話し言葉で）

## 【本題】
**テロップ:** （根拠・解説の要点。15文字以内）
**ナレーション:** （具体的な根拠や説明を2〜3文。話し言葉で）

## 【行動喚起】
**テロップ:** （保存・フォロー等を促す1行。15文字以内）
**ナレーション:** （行動を後押しする1〜2文。話し言葉で）
```

保存後、「テロップとナレーション.md に保存しました」と伝える。

### 注意点
- [ジャンル] / [VIDEO_NAME] は実際の値に置き換える
- 動画に映る商品・シーンと整合性のある内容にする
- テロップは短く端的に（15文字以内を目安）
- ナレーションは話し言葉・です/ます調で自然に
