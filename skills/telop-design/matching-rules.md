# 映像→パターン マッチングルール

映像の特徴を分析し、patterns.mdの10パターンから最適なものを選択するための判定ルール。

---

## 判定フロー

```
映像分析 → ① 明暗判定 → ② 被写体判定 → ③ 空きスペース判定 → ④ 雰囲気判定 → パターン決定
```

---

## ① 明暗判定（テキスト色を決める）

| 映像の明暗 | テキスト色 | 候補パターン |
|-----------|----------|------------|
| 暗い（黒背景、夜、暗めの室内） | 白系 | P1, P2, P3, P4, P6, P8 |
| 明るい（白背景、昼、明るい室内） | 暗色系 | P2, P5, P7 |
| 中間・複雑（部分的に明暗混在） | 白＋シャドウ or 帯 | P1, P3, P8 |

---

## ② 被写体判定（配置位置を決める）

| 被写体の位置 | テロップ配置 | 候補パターン |
|------------|-----------|------------|
| 中央に商品 | 下部 or 上部（商品を避ける） | P1, P3, P8 |
| 上部に人物の顔 | 下部 | P1, P8 |
| 下部に人物 | 上部 | P3 |
| 中央に人物（全身 or 上半身） | 端（左右） | P4, P7 |
| 被写体が小さい / 余白多い | 中央 | P2, P6 |
| 画面全体を埋めている | 帯付き | P8 |

---

## ③ 空きスペース判定（配置を確定する）

| 空きスペース | 最適配置 | 候補パターン |
|------------|---------|------------|
| 上部が空き | 上部配置 | P3, P5 |
| 下部が空き | 下部配置 | P1, P8 |
| 左右どちらかが空き | 端配置 | P4, P7 |
| 四隅のみ空き | 隅配置 | P5, P7 |
| ほぼ空きなし | 帯で強制確保 | P8 |

---

## ④ 雰囲気判定（最終決定）

| 映像の雰囲気 | 最適パターン | 理由 |
|------------|-----------|------|
| **高級感・ラグジュアリー** | P2 or P4 | 余白・細字・装飾なしで品格を出す |
| **力強い・訴求力** | P1 or P3 | 太ゴシック大文字でインパクト |
| **清潔感・スキンケア・ナチュラル** | P5 or P2 | 細字ミニマルで清潔さを表現 |
| **和風・伝統** | P4 or P10 | 縦書き・明朝・筆で和の品格 |
| **モード・おしゃれ・ファッション** | P7 or P2 | 英字セリフで洗練された印象 |
| **カジュアル・ポップ・食品** | P9 or P6 | 色ベタ・太字で親しみやすさ |
| **セール・キャンペーン** | P6 or P9 | 数字を巨大に、色で目を引く |
| **万能・迷った時** | P8 | 帯テキストはどんな映像にも合う |
| **シネマティック・映画的** | P1 or P4 | 下部太ゴシック or 端縦書きで映画字幕感 |
| **エモーショナル・感動** | P4 or P2 | 縦書き明朝 or 中央細明朝で情緒 |

---

## シーン役割別の推奨パターン

| シーン役割 | 推奨パターン | 理由 |
|----------|-----------|------|
| **フック（1枚目）** | P3, P1, P6 | 最初の1秒で目を引く。大きく・太く・インパクト重視 |
| **本題（中間）** | P1, P4, P8, P7 | 情報を伝える。読みやすさとデザインのバランス |
| **行動喚起（最後）** | P8, P5, P2 | 控えめだが明瞭。視聴者に次のアクションを促す |

---

## クリップ本数別の組み合わせ方

### 2本の場合
```
Scene 1（フック）:    インパクト系（P1, P3, P6）
Scene 2（CTA）:       控えめ系（P5, P2, P8）
```
2本は強→弱のコントラストを意識する。同系パターンにしない。

### 3本の場合（標準）
```
Scene 1（フック）:    インパクト系（P1, P3, P6）
Scene 2（本題）:      情報系（P4, P7, P8）
Scene 3（CTA）:       控えめ系（P5, P2, P8）
```

### 4本以上の場合
```
Scene 1（フック）:    インパクト系
Scene 2〜N-1（本題）: 情報系を交互に。同じパターンを連続させない
Scene N（CTA）:       控えめ系
```
本題が複数ある場合は「P4縦書き → P1横書き → P7二層」のように配置・向き・フォントをシーンごとに変える。

## 組み合わせ例（3本・ジャンル別）

### 高級・美容系
```
Scene 1（フック）: P3 上部大文字ヘッドライン — 太ゴシック96px白
Scene 2（本題）:   P4 縦書き端配置 — 明朝72px白
Scene 3（CTA）:    P5 左上ミニマル — 細ゴシック48px暗色
```

### 食品・カジュアル系
```
Scene 1（フック）: P6 中央インパクト数字 — 極太140px赤
Scene 2（本題）:   P1 下部左寄せ太ゴシック — 太ゴシック84px白
Scene 3（CTA）:    P8 下部帯テキスト — 帯付き52px白
```

### シネマティック・感動系
```
Scene 1（フック）: P1 下部左寄せ太ゴシック — 太ゴシック84px白
Scene 2（本題）:   P2 中央ブランドロゴ — 細明朝64px白
Scene 3（CTA）:    P4 縦書き端配置 — 明朝72px淡色
```

### モード・ファッション系
```
Scene 1（フック）: P7 英字+日本語二層 — セリフ56px+ゴシック32px
Scene 2（本題）:   P3 上部大文字ヘッドライン — 太ゴシック96px白
Scene 3（CTA）:    P2 中央ブランドロゴ — 細明朝64px
```

## 判定の禁止事項

1. **前回の動画と同じパターン組み合わせを使わない**
2. **3シーンすべて同じフォントweight（太さ）にしない** — 強弱のコントラストが必須
3. **P5（小サイズ）をフックに使わない** — 最初のシーンは大きくインパクトを出す
4. **P6, P9を高級系映像に使わない** — カジュアル感が品格を壊す
5. **メインテロップを64px未満にしない** — 動画は5秒で消える。小さいと読めない
