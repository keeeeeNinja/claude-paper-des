## フォント

### 使用可能なフォント一覧

#### システムフォント（常に使える）

| フォント名 | CSS指定 | 特徴 | 向いている動画 |
|----------|---------|------|-------------|
| ヒラギノ角ゴシック | `"Hiragino Sans, sans-serif"` | 日本語標準ゴシック。クリーンで読みやすい | スキンケア・食品・ファッション全般 |
| ヒラギノ明朝 | `"Hiragino Mincho ProN, YuMincho, serif"` | 品のある明朝体 | 高級品・和食・ブライダル |
| Georgia | `"Georgia, serif"` | 英字セリフ。上品でモード感 | ファッション・海外ブランド |
| Arial / Helvetica | `"Helvetica Neue, Arial, sans-serif"` | 英字ゴシック。スポーティ・モダン | スポーツ・テック・カジュアル |

#### weight（太さ）の使い分け

| weight | 印象 | 使いどころ |
|--------|------|----------|
| 200 | 極細。繊細・高級 | ブランドコピー・CTA |
| 300 | 細。上品・ミニマル | 高級系の本題・行動喚起 |
| 400 | 普通。自然 | 補足テキスト |
| 600 | 中太。しっかり読める | 汎用メインテロップ |
| 700 | 太。力強い | フック・訴求系 |
| 800 | 超太。インパクト | フック最優先シーン |
| 900 | 極太。圧倒的インパクト | セール・数字訴求 |

#### 組み合わせルール
- **3シーンで異なるweightを使う**（例: 900 → 300 → 600）
- **1本の動画で使うフォントは最大2種類まで**（ゴシック+明朝 or ゴシック+セリフ）
- 細字（200-300）はフォントサイズを大きめ（64px以上）にしないと読みにくい

## 色

### テキスト色パレット

#### 白系（暗背景向け）
| 色 | HEX | 使いどころ |
|----|-----|----------|
| 純白 | `#FFFFFF` | コントラスト最大。力強い訴求 |
| オフホワイト | `#F5F0E8` | 温かみのある白。食品・スキンケア |
| クリーム | `#F5E6C8` | 高級感ある淡いゴールド。ブライダル・和 |
| アイスホワイト | `#EAF4F8` | 清涼感。飲料・スポーツ |

#### 暗色系（明背景向け）
| 色 | HEX | 使いどころ |
|----|-----|----------|
| ほぼ黒 | `#1A1A1A` | 最もミニマルで品がある |
| チャコール | `#2A2A2A` | 黒に近いが柔らかい。スキンケア・ミニマル |
| ダークグレー | `#3D3D3D` | 読みやすい暗色。万能 |
| ウォームダーク | `#2C1810` | 温かみのある暗色。食品・和食 |

#### アクセントカラー（特定ジャンル用）
| 色 | HEX | ジャンル |
|----|-----|---------|
| ゴールド | `#C9A84C` | 高級品・ブライダル・日本酒 |
| ローズゴールド | `#B5828C` | コスメ・美容・女性向け |
| ネイビー | `#1B2A4A` | 時計・スポーツ・信頼感 |
| テラコッタ | `#C4603A` | 和食・自然派・秋冬 |
| セージグリーン | `#7A9E7E` | 自然・オーガニック・ヘルス |
| レッド | `#E53935` | セール・キャンペーン・食品 |

### 背景・帯の色

| 用途 | CSS値 |
|------|------|
| 黒帯（標準） | `rgba(0, 0, 0, 0.45)` |
| 黒帯（強め） | `rgba(0, 0, 0, 0.65)` |
| 白帯（明背景） | `rgba(255, 255, 255, 0.75)` |
| ブランドカラー帯 | `rgba(R, G, B, 0.85)` |

## 装飾

### 装飾なし（最もミニマル）
```js
// 装飾要素ゼロ。フォント・色・配置だけで勝負
// P2, P5向け
```

### テキストシャドウ（標準）
```js
// 柔らかいシャドウ（読みやすさ確保）
textShadow: "0 2px 16px rgba(0,0,0,0.4)"

// 強めのシャドウ（複雑な背景向け）
textShadow: "0 2px 24px rgba(0,0,0,0.7)"

// 白テキストの場合のみ使用。暗色テキストには使わない
```

### 下部帯（P8向け）
```jsx
// Telopコンポーネントの構造を変更してdivで帯を作る
<AbsoluteFill style={{ justifyContent: "flex-end", alignItems: "stretch" }}>
  <div style={{
    backgroundColor: "rgba(0,0,0,0.45)",
    paddingTop: 24,
    paddingBottom: 32,
    paddingLeft: 48,
    paddingRight: 48,
    width: "100%",
  }}>
    <div style={{ /* textStyle */ }}>
      {text}
    </div>
  </div>
</AbsoluteFill>
```

### テキスト縁取り（P6向け）
```js
WebkitTextStroke: "2px #FFFFFF",  // 白縁取り
// または
WebkitTextStroke: "2px #000000",  // 黒縁取り
// セール・インパクト系のみ。高級系には使わない
```

### アンダーライン
```js
// シンプルなアンダーライン
textDecoration: "underline",
textUnderlineOffset: "8px",
textDecorationThickness: "2px",

// カスタムアンダーライン（borderBottomで実装）
borderBottom: "2px solid rgba(255,255,255,0.6)",
paddingBottom: "4px",
```

### ボックス囲み
```jsx
// テキストの周りに枠線
<div style={{
  border: "1px solid rgba(255,255,255,0.6)",
  padding: "12px 24px",
  display: "inline-block",
}}>
  {text}
</div>
```

### グラデーションテキスト
```js
// ゴールドグラデーション（高級系）
background: "linear-gradient(135deg, #C9A84C 0%, #F5E6C8 50%, #C9A84C 100%)",
WebkitBackgroundClip: "text",
WebkitTextFillColor: "transparent",
backgroundClip: "text",
```

## 装飾の使用ルール

| 雰囲気 | 許可する装飾 | 禁止する装飾 |
|--------|------------|------------|
| 高級・ブランド | なし、シャドウ（控えめ）、グラデーション | 縁取り、ベタ帯、ボックス |
| スキンケア・清潔感 | なし、シャドウ（極控えめ） | 縁取り、ベタ帯 |
| シネマ・感動 | シャドウ（強め）、アンダーライン | 縁取り、カラー帯 |
| カジュアル・食品 | ベタ帯、縁取り、ボックス | ― |
| セール・キャンペーン | 縁取り、カラー帯、グラデーション | なし（装飾が必要） |

**大原則: 装飾は引き算。迷ったら「なし」を選ぶ。**
